  1. git init
  2. git add .
  3. git commit -m "Ready for Windows build"
  
  Then create a repo on GitHub and push:
  
  4. git remote add origin https://github.com/YOUR-USER/YOUR-REPO.git
  5. git push -u origin main
  
  6. Go to GitHub → Actions tab → Wait for build
  7. Download the .exe file
  8. Run: ./create-client-package.sh
  9. Send the ZIP to your client!

Option B: Using Windows PC

  1. Copy project to Windows
  2. npm install
  3. npm run build:win
  4. Run: ./create-client-package.sh
  5. Send ZIP to client!
