                        üé® CUSTOMIZE ICONS (OPTIONAL)
================================================================================

The system includes placeholder icons. To use your own logo:

1. Create a 512x512 PNG of your logo
2. Save it as: assets/icons/icon-512.png
3. Run: node create-icons.js
   (Or on Linux with ImageMagick: ./create-icons.sh)
4. Rebuild: npm run build
