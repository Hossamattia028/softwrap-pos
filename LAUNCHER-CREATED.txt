================================================================================
                          TROUBLESHOOTING
================================================================================

Launcher not appearing?
-----------------------
1. Wait 10-20 seconds (Ubuntu needs time to refresh)
2. Log out and log back in
3. Or run: ./create-desktop-launcher.sh again

Launcher won't open app?
------------------------
1. Make sure you're in the project directory
2. Test manually: cd /media/hossam-attia/data/desktop_application/main_pos && npm start
3. Check for errors in terminal
4. Kill any zombie processes: pkill -f "electron.*softwrap"

Icon not showing?
-----------------
1. Check icon exists: ls -la assets/icons/icon.png
2. Refresh icon cache:
   $ gtk-update-icon-cache ~/.local/share/icons/hicolor/ -f
3. Re-run: ./create-desktop-launcher.sh
